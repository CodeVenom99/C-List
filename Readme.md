总结：

1.创建头节点有两种方式

（1）利用返回值，给头结点赋值，容易理解

（2）函数传参。采用这种方式的时候，因为需要改变的是指针的值，所以需要注意形参必须是指向结构体指针的指针，否则传入的实参是不会改变的

2.创建出来的头节点是指向头结点的

头结点是空结点，（就是创建头节点的时候头结点的数据域是可以赋任意值的，指针域应为为NULL）这个结点在后续的使用中，是不做存储数据用的，也无法当做存储数据使用，应为如果存储了数据，这个结点在delete_list的处理就会比较麻烦

2.创建链表

（1）思路是先创建一个临时的指针temp，指向一个新开辟的结构体空间，为这个结构体空间赋值，（数据域，指针域=NULL）完成这个操作后将p->pNext = temp，最后p在后移

（2）这里的形参是NODE* p，在函数体内操作的p与传入的实参p是一个值，形参p是一个指向结构体的指针，对其指向的结构体修改，当然会改变实参p所指向的结构体，但不会改变实参p的值。这个要与	1.（2）结合理解

3.删除结点

（1）对于实参p在迭代的时候，最后要指向pos-1的位置，指向pos就没法删除了。for (int i = 0; i < pos-1; i++)	p = p->pNext;p最后指向pos-1

（2）对于被删除的结点，需要用一个临时指针将其保存，方便最后free，这里要注意，free只会将这个结构体的内存释放，不会释放这个结构体中指针域的指向
